Velocity is a Windows-focused Roblox script executor built to run Lua scripts through a clean desktop interface. It brings together execution tooling, a large built-in script hub, and an auto-update system so the app can stay current without extra manual work.

## FAQ

**Is Velocity limited to Windows?**  
Yes. It is described as a desktop executor for Windows 10/11 only.

**Can it update itself?**  
Yes. An auto-update engine is included so compatibility can be improved as newer builds are released.

**What sets it apart from a simple script runner?**  
Velocity adds a larger script hub, batch execution, queue management, and a built-in debugger instead of stopping at a basic launch screen.

**Is there any account risk?**  
Use is subject to Roblox terms and your own decisions. Review platform rules and understand the implications before running any third-party scripts.

**Are scripts stored locally?**  
The workflow includes persistent queue support and local application data handling, so scripts and settings may be kept on the machine depending on your setup.
